# Learning UI/UX Project

This project was created as part of the CodeAlpha UI/UX Internship.

## Project Overview
Learning is a mobile learning application designed for students to access courses, video lessons, and track learning progress.

## Features
- Login Screen
- Course Dashboard
- Course Details
- Video Learning Screen
- User Profile

## Tools Used
